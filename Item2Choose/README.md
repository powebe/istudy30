==使用說明：==
  利用 broadcast "OnStage/OffStage" 啟動/結東
  參數：lcRoleX, lcRoleY, 設定出現位置
              lcRoleBetweenX 設定物的間隔
              gStageMax 設定出現幾個選項
  結果：用 gSelectedItem 來抓所選到的東西項目
  註：gOnStageSprite、gRoleFirst，都要設成 global (for all sprite)
